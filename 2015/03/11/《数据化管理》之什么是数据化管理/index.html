《数据化管理》之什么是数据化管理

发表于 2015-03-11 | 分类于 读书笔记

最近阅读《数据化管理：洞悉零售业及电子商务运营》这本书，特意整理，欢迎交流指正。

1. 数据化管理的意义：基于业务的应用是数据化管理的核心，数据化管理与数据分析最大的不同就是数据化管理将数据分析应用在了业务层面才具有意义。

  1. 量化管理：通过对工作进行量化就可以对日常业务进行一种量化管理。

  2. 最大化销售业绩，最大化生产效率：将正确的数据分析结果用在最为实际的业务上面，使得产生效益，持续不断的生产效应就可以称之为「数据化管理」

  3. 有效的节约企业各项成本和费用：通过数据化管理，可以预知并追踪企业的成本，从而达到节约成本和费用的目的。

  4. 组织管理、部门协调工具：针对不同部门进行不一样的数据指标，制定不一样的数据化管理体系，提高不同部门之间的协调能力和效率。

  5. 提高企业管理者决策的速度和正确性：正确的数据分析不应定可以支撑制定战略，数据化管理可以为管理层决策提供科学的支持。

2. 数据化管理的四个层次

  1. 业务指导管理：包括销售、人力资源、生产、财务、客服等业务单元。

  2. 运营分析管理：简单来说就是讲对人、货、场、财的分析管理。

  3. 经营策略管理：通过对各个经营环节进行对应的数据分析达到制定或修改策略的目的。

  4. 战略规划管理：包括宏观经济分析、行业环境分析、经营环境分析、内部资源分析、企业竞争分析、战略目标规划管理、战略可操作性评估等。

3. 数据化管理流程图

  1. 分析需求：包括收集需求、分析需求、明确需求。常常使用的方法就是访谈、市场调查、走访专家等。具体实现就是利用思维导图进行5W2H分析。

  2. 收集数据：收集的途径包括公司数据库、公开出版物、市场调查、互联网、购买专业公司数据等。

  3. 整理数据：把收集到的结构杂乱的数据进行标准化的处理。

  4. 分析数据：利用分析软件结合业务需要对数据进行分析，得出数据报告。

  5. 数据应用：数据话管理的目的就是应用，应用才是整个数据化管理的重点。
