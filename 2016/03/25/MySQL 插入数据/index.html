MySQL 插入数据

发表于 2016-03-25 | 分类于 学习札记 |

使用 MySQL 数据的时候可以通过使用 INSERT 这个语句来进行插入操作，给数据表插入数据可以有多种方式，可以插入整行数据；可以对一行当中提定的列插入数据；还可以插入某些查询出来的结果数据；还可以复制其他表格数据到数据表里面。

注：INSERT 语句不仅适合用于 MySQL ，同时适用于几乎所有类型的数据库，只是部分类型数据库操作语句有点不一样。

在插入数据之前首先就要选择进入对应操作的数据表，在连接数据库成功的情况下使用 use database_neme 语句进入相应的数据库进行操作，比如需要操作数据库名为 WP 这个数据库，那么在命令行数据入 use WP ,如下显示即可成功切换数据库。
