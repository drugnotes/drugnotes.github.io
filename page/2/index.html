SEO 中的爬行、抓取、索引和收录

发表于 2017-07-31 | 分类于 互联网 |

搜索引擎的基本原理：首先使用搜索引擎蜘蛛，沿着网站的 URL 「爬行」，每爬一个 URL，就保存该 URL 到待抓取库，然后访问待抓取库中的 URL，对该页面的内容进行「抓取」，解析 HTML 的内容，系统判断该页面内容的质量，考虑是否值得「收录」，如果值得收录，那就将该 URL 在搜索引擎系统中建立「索引」。

开头的这段话分别介绍四个动作，那就是「爬行」、「抓取」、「收录」和「索引」，也是今天这篇文章讨论的内容？

什么是爬行？
爬行是指搜索引擎的爬虫机器人，我们称之为「搜索引擎蜘蛛」，从已知网站的URL 出发，沿着网站的每个URL地址，新发现的 URL 进行不断的爬行，并将新发现的 URL 地址保存在待抓取的地址库中。

什么是抓取？
抓取是指访问在爬行中获取到的待抓取地址库中的 URL，并且将这些 URL 的页面进行解析，将其解析到的内容（HTML 代码）保存在数据库中。

什么是索引？
索引是指将前面解析出来，并且认为是「有价值内容」的页面，在数据库中建立对应关系，形成索引库。当用户在搜索引擎中针对某个关键词进行搜索，根据关键词和索引的关系，将页面反馈给到用户，这也是搜索引擎的基本工作原理。也就是说，只有添加了索引的 URL 才能被用户在搜索引擎中搜索出来。

什么是收录？
其实收录的意思就是指 URL 的信息在搜索引擎的索引库中建立了索引，所以收录和索引是一样的意思，索引是指机器索引，对于网页来说，是被机器收录。

nofollow 属性是什么？
在网站页面中，通常会存在一些链接，我们可以对这些链接设置nofollow属性。nofollow属性的意义就是告诉爬虫蜘蛛：这个 URL 地址不要继续爬行，就当作不是链接来处理。为什么会存在 nofollow属性呢？最根本的作用就是：对于外链，尤其是质量差外链（比如付费广告），站长需要添加 nofollow属性来减少网站 RP 值的外流，保持网站的权重。虽然爬虫蜘蛛在某个页面爬到某个 URL 有nofollow属性，停止爬行。但是在别的地方再遇到这个 URL 的时候，并且没有nofollow属性，这个 URL 还是会被蜘蛛爬行，抓取，甚至是建立索引，将其收录。

noindex 标签是什么？
在网页头部信息中，添加meta noindex标签信息，告诉搜索引擎，我这个网页不要添加索引，不要让用户在搜索引擎中搜索出来。具有noindex标签的页面，搜索引擎蜘蛛是会爬行并且抓取的，因为爬行和抓取是在索引的前面的，只是到解释到该页面的HTML内容的时候，发现存在noindex标签，只会将其内容保存起来，但是不会对页面建立索引，用户也就不能在搜索引擎搜索到该页面。

robots.txt 是什么？
robots.txt 亦称之为 robots 协议，一般存在于网站的根目录下，一般直接访问「www.xxxx.com/robots.txt」都能看到 robots.txt 的内容。robots 协议会告诉搜索引擎蜘蛛，该网站下哪些页面可以抓取，哪些页面不可以抓取。robots 协议并不是所有的搜索引擎都遵守，但是主流搜索引擎都是遵守 robots 协议的。所以说，robots 协议并不是行业规范，也是一种约定俗称。

最后几点补充
无论是nofollow 、noindex还是 robots 协议，当然还有其他的一些搜索引擎规则，这些属性、标签和协议，并不是一开始，所有的搜索引擎都遵守。比如说，对于 robots 协议，直到 2008 年，Google、雅虎、百度这些主流的搜索引擎才一致承认这样的协议，承诺自家的搜索引擎蜘蛛将会遵守 robots 协议上面的内容取爬取网站的内容。所以，搜索引擎就是一个不断完善，不断进步的产品，无论是技术还是准则，都是一个迭代的过程。

佛教与中国早期贸易

发表于 2017-07-19 | 分类于 学习札记 |

中国与外部世界交往的方式中，最重要的是贸易。而最初的贸易，只是官府的副产品，由于官府一直对商人存有疑虑，贸易大多为外国人所控制。早期沿着『丝绸之路』，将中国丝绸卖到印度、波斯，甚至是罗马。

在西方流入中国的商品则是相反，大部分流入中国的商品都是用来满足统治精英的好奇心稀有物品，并且很多一部分带有文化元素，其中最重要的就是佛教。

茶叶与白糖随着佛教的兴起增加了需求，椅子也随着佛教从西域传入中国，纸张改良和印刷术发展也受到佛教信仰的重要影响。

对于外来的珍稀商品，最后会被放在佛像前或寺庙中，用于供奉朝拜。这些珍惜物品最首要的就属『七宝』：黄金、白银、天青石、水晶或石英、珍珠、红珊瑚、玛瑙或者珊瑚。出了来自异域的『七宝』之外，在佛教信仰中用到的焚香各种香料也是从印度进口而来。

除此之外，佛教信仰也在文化层面改变人类的思考和观念，形成新的消费和经营观念，在某种程度也改变了未来的商业社会。因此而知，信仰不仅可以带来文化的改变，还会改变商业社会，人类的消费观念。

之道今天，还有人利用信仰销售，甚至是利用人们的信仰骗取财产。最典型的例子就是有人利用微信公众号，进行信仰传播，开发虚拟的祭奠、放生的小游戏，骗取老年人的钱。这样的事情并不是个例，而是屡见不鲜。

当然，这并不是证明信仰，尤其是佛教的不好，甚至完全没有这个意思，仅仅是用来阐述过去佛教是怎么影响国内的贸易，仅此而已。

互联网的本质：联·互·网

发表于 2017-07-18 | 分类于 互联网 |

互联网作为这个时代最重要的技术革命，其本质就是三个字：联·互·网。

第一个关键字：联，也就是联结。最早，互联网将机器与机器，人与机器，人与人以及人与社会连接起来。使得任何的物品不再是孤立，任何的人也不再是孤立的个体。在未来，物联网的发展，物与物、物与人直接的联结，物品也会变得更加的智能。

第二个关键字：互，也就是互动。过去，互联网不仅将人与机器连接，并且能与机器进行互动。但是，人机之间的互动是非常弱，存在很大的限制。随着智能手机的出现，移动互联网技术的发展，4G 网络的普及，人机之间的互动变得更加便捷和多样化。继续发展出现人与人、人与社会的大规模互动，产生了 SNS，大量的社会协作通过互联网进行。

第三个关键字：网，也就是结网。有了互联网的联结和互动，互联就是一张很大的网，这张联结和大量的人和物。通过这张网，人与人之间完成协同、分工和合作。除此之外，信息、物和人都在线了，人和人、人和信息、人与物之间的互动才会越来越丰富，最后交织成越来越繁密的网络，可以用更高效的方法去完成原来很难实现的事情。

短句

发表于 2017-07-17 | 分类于 生活随笔 |

近日，在书店再次读到阿城的作品，《闲话闲说：中国世俗与中国小说》。书的前半部分，讨论中国世俗文化，后半部分，讨论中国小说。

最早读阿城的作品是《棋王 树王 孩子王》，阿城精简有力的文字把我迷住，越发喜欢。书中的大多内容，未能透彻理解，但读这样的文字，身心愉悦。

早就听闻阿城是国内作家中，动词用的最好的。汪曾祺这样评论：

读了阿城的小说，我觉得，这样的小说我写不出来。 我相信，不但是我，很多人都写不出来。

汪曾祺在短句写作方面，也是一等一的高手，可谓是知己相逢。

过去，还读过王烁翻译的《经济学人》写作法，又名《有效写作十三篇》。有效写作，指在写出来，把事说清楚，让人理解。依稀记得，书中建议，尽可能少用『的』、『了』、『呢』等语气词，每当发现，删除也不影响原文意思。

像汪曾祺、阿城、王烁的短句写作能手，并不多见，这需要刻意练习，方可习得。

购物网站为什么要销售购物卡

发表于 2017-06-26 | 分类于 互联网 |

第一个推广购物卡的电商网站应该是 Amazon 吧，并且那么多年来都没有放弃过。我很难理解为什么这些购物网站会愚蠢到发行这样的购物卡，不仅使用不方便，费时费力，而且在商品选购时会设立诸多的限制。Amazon 这种形式不仅在中国没有消失，而且很多国内电商网站也开始模仿起来，销售起购物卡，很长一段时间我不能理解。

直到今天，我收到公司为我提供的生日福利——两张某购物网站的购物卡。虽然价值不是很高，但是我也觉得蛮开心，起码还是有福利的。让我开始思考自己过去对购物卡的看法。我想，这些购物网站推广购物卡业务，压根就是不为了让最终消费者去购买然后使用，更多的是用来送礼或者作为企业员工福利。还有很多公司作为促销的工具。比如投保 10000 块钱的保险，就可以或 2000 块钱购物卡。这样的促销活动看上去很诱惑，实际就是一种促销手段，最后羊毛还是处在羊身上。企业采购这些购物卡付出的成本比实际面额要小得多，至于购物网站为什么愿意低价销售购物卡，因为这些购物卡限定品类，就相当于用这些购物卡去销售滞销的商品，降低库存成本。

除此之外，购物卡还会衍生例外的产业链，那就是购物卡回收。网络上，专门有人回收、倒卖这些购物卡，从中牟利，这让我想起了月饼券的故事：

公司以6折~7折的价格从厂家团购月饼票发给员工；员工以4折~6折的价格把月饼券卖给黄牛；黄牛把倒卖不掉的月饼券以4折~5折的价格卖回印券的厂家。月饼券转了一圈，厂家实际上却并未卖出一个实实在在的月饼，月饼票于是演变为一种“期货类的金融商品”。厂家卖了一张自己印的“纸”，就可以赚取相当于券面面值2折的真金白银。

讲了这么多，并不是有心去黑这些商家，只是想要说明白一个道理：很多我们认为不会成立的商业逻辑或者模式，但是却实实在在的存在着，仅仅是因为我们站的角度不同而已。比如面对购物卡，如果仅仅站在消费者的角度，是无法知道这样的商业模式久经不衰的原因。

究竟需要付出多大的努力才能成为专家

发表于 2017-06-20 | 分类于 生活随笔 |

回过头来，发现自己已经在职场上走过两年，虽然相比过去刚出校园实习的时候，无论是专业技能还是职场经验，都有所提高。但是放眼望去，很多优秀的同龄人，先有的行业地位和职业成就远远高于自己，不得让人反思自己两年的职场生涯。

现在我们都喜欢斜杠青年，也都想成为那样的人。但是自己每天都很努力，每天都很忙，但是效果总是不佳。难道那些斜杠青年就真的比我们有更多的时间和精力。其实不尽然，反观自己的生活和工作，生活并没有多出色，工作虽然不是很差，但还是有很大的提升空间。

最近在反思该不该无休止的加班，加班意味着工作没有做完，无休止加班意味着工作永远做不完。其实，每天八小时的工作时间，足以用来应付工作所有的工作，只是在那八小时当中，我们是否尽情的投入到工作当中，抑或是混时间，或者把工作时间用来做别的事情。工作时间，很多干扰因素会影响到我们的工作效率，针对琐碎的事情，我们是不是安排在特定的时间处理，甚至是下班之后再处理。

前几天，在「得到」上面听到吴军老师的《硅谷来信》，里面提到人工智能为什么进步这么快，就是因为人工智能可以专心去做一件事情，把一件事情做完之后再去做别的事情，所以效率非常高。

在目标和时间管理上，自己确实存在很多的问题，假如这些问题没有很好的解决，将会在未来的职场生涯给自己带来很大的负面影响，很多时候决定一个人未来未来的并不是需要付出多大苦劳，而是看你贡献多少功劳。

谈谈微信指数

发表于 2017-03-24 | 分类于 互联网 |

什么是微信指数？
昨天晚上，微信推出微信指数，简单来说，微信指数是微信官方提供的基于微信大数据分析的移动端指数。如果对百度指数、微博指数、淘宝指数等有了解的人应该都知道指数这个东西有什么用途，可以用来检查某个关键词在某个时间点的被关注的情况以及舆论的走向。

与其他的指数类似但也有不同，微信指数整合了微信上的搜索和浏览两种行为数据。相对而言，微信上面搜索功能比较简单，虽然从去年年底开始，微信开始布局微信搜索框，准备抢占搜索入口。但是看目前的情况，直接通过微信搜索的人还比较少，所以目前微信指数很大的数据占比还是来自浏览的数据。

为什么要做微信指数？
过去 PC 时代，以搜索为入口的 Google 和百度分别有 Google trends 和百度指数；淘宝有针对消费查询的消费指数（现已下线）；微博有属于社交和舆论的微博指数；知乎最近也在内测技术话题和回答的知乎指数；所以，作为体量最大，用户时间时间最长的微信，也能根据用户搜索和浏览内容去做微信指数这个产品。

除此之外，上线微信指数也就意味着微信已经掌握流量和内容的入口。

微信指数有什么值得期待？
目前上线的微信指数，只是一个非常简单的版本，收录的词库远远不够，只提供某个关键词的趋势，不能对相关的关键词对比，没有用户画像和需求图谱等。除此之外，在技术层面，明显是存在一些问题，很多关键词在某天突然上升几十倍甚至上百倍，而且只能保留一天，作为微信这么大体量的产品，产品数据不会出现这么大的波动，这一定就是技术的问题。

简单来说，现在的微信指数，就是一个玩具，说什么读懂趋势、检测舆情动向、洞察用户兴趣、助力精准营销，统统都是吹嘘。

那么，微信指数是不是就不值得期待呢？那倒不是？

如果微信指数后期继续增加新功能，将用户画像，关键词需求图谱、时间点分布等功能上线之后，微信指数能在内容运营上面提供很大的帮助。过去的微信公众号运营者看到的都是订阅了自己公众号的用户行为数据，看不到整体用户的行为数据，很容易就会导致在自己的误区中生产内容。除此之外，内容生产者通过微信指数，更容易把握内容方向。假设这么一个场景：某个自媒体正在发愁今天的推送内容的时候，只需要到微信指数上面看看哪些关键词最高，然后通过这些关键词去找文章，再从这些文章中截取一部分文字，拼凑起来就成了一篇热点文章啦。

写到这，我头脑里面突然冒出了一个场景：某天晚上，我打开微信订阅号，看到一排下来的推送全都是针对某个关键词，不过好像现在的公众号推送的内容也是差不多耶。

当然，微信指数的功能不仅仅在内容运营上提供参考，还可以有很多种可能，可以有很大的想象空间。
