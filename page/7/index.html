为什么总是抓不住机会

发表于 2016-02-28 | 分类于 生活随笔 |

周末回了趟学校，晚上几个同学散步的时候，走到了学校附近的一个新开放的楼盘里面。原本楼盘的那块地是一片长满荒草地小山坡，去年年初的时候开发商就开始推平这座小山，不到一年的时间，楼房就被建了起来，而且白天的时候来看房的人特别的多。我们在里面散步的时候，不少的同学都在叹气：「如果自己有钱，之前能把这地买了那就发了」。当然，这是一句玩笑，也是一句自嘲。

面对着现实生活的压力，我们尝试在给自己的生活寻找一个突破的地方，或者去找一些可以让自己快速成长，快速发财的机会。当然，我们遇到了很多的”机会“，只是往往抓不住。

在 过去的前十年，我们没有抓住互联网的爆发期；在淘宝快速成长的时候，我们没有抓住淘宝卖家的机会；在 08 年楼长高涨的时候，我们没有抓住机会；在微博时代，我们没有抓住机会；在微信时代，我们没有抓住机会；在互联网金融风口期，我们没有抓住机会；甚至是在当下很多的风口上面，我们都没有抓住机会，我们都只是一个旁观者，只能眼睁睁的看着这些机会被别人抓住了，并且在这个行业里面发展起来了。

或者我们都还没有明白为什么抓不住这些机会的时候就已经过了风口期了，或者说很多东西，到了现在还是不能明白为什么当时没有抓住这个机会。

在我看来，其实并不是你没有抓住这些机会，只是觉得这些所谓的”机会”对于自己来说根本就不是机会，那只是对于特定人的机会。

其实行业与行业之间都是有一堵很厚的强，墙外的人根本不知道墙内里面最真实的情况，甚至连墙内也就是只有少部分的人真实准确的理解墙内的情况。在加上这些人具备了在这堵墙里面生存的根本技能，所以他们就能很好的抓住这些机会。而那些墙外的人，更多的只是充当一个旁观者，看到的也就只能是最热闹的那部分，所以也就不能很好的理解机会，更加没有在墙内生活的根本能力。

李笑来之前在写一本英语学习的书的时候，使用到了关于程序的东西来帮他整理词汇，正好在很多年前，李笑来就学习了编程相关的东西，当时学习的时候，他并不知道这些编程的能力对他的未来有什么好处，也就学了。所以可以知道，出那本书的为什么是李笑来而不是其他的新东方老师，我想新东方里面肯定有比李笑来的英语好的人吧。

其实我总结了一下，抓不住机会的根本原因就是你们有对这个行业有足够的知识和技能，其实说白了就是平时没有学习。

今天要写的就是这么多，也就是这么简单，但并不是每个人都懂，包括我自己。

首先就是现在项目文件上面新建一个.txt 文件，文件的内容随意即可。

打开（fopen()）文件

fopen($filename, $mode) 是用来打开文件，其中包含了两个参数 $filename 就是要打开文件的文件名，$mode 是打开的模式，打开文件的模式有很多，具体可以这样理解。

r 打开文件为只读。文件指针在文件的开头开始。

w 打开文件为只写。删除文件的内容或创建一个新的文件，如果它不存在。文件指针在文件的开头开始。

a 打开文件为只写。文件中的现有数据会被保留。文件指针在文件结尾开始。创建新的文件，如果文件不存在。

x 创建新文件为只写。返回 FALSE 和错误，如果文件已存在。

r+ 打开文件为读/写、文件指针在文件开头开始。

w+ 打开文件为读/写。删除文件内容或创建新文件，如果它不存在。文件指针在文件开头开始。

a+ 打开文件为读/写。文件中已有的数据会被保留。文件指针在文件结尾开始。创建新文件，如果它不存在。

x+ 创建新文件为读/写。返回 FALSE 和错误，如果文件已存在。

黑客与画家

发表于 2016-02-21 | 分类于 读书笔记 |

这是一本非常经典的书籍，作者是硅谷创业之父——Paul Graham，译者是我非常喜欢的博客作家阮一峰。这本书不仅作者写得好，译者也很给力，不会觉得读起来文字显得很生硬。

只是简单的看题目，读者可能很容易会被误导为这本书讲的就是黑客与画家。其实不然，《黑客与画家》这是这本书其中的一篇文章而已，当然作者在书中的很多地方提到黑客，并且认为黑客就是技术创新发展的源头。之所以会有这样的赞誉，其中的原因就是作者就是一个黑客，另外一个原因是事实确实如此。

在这本书中，作者写了很多关于互联网，财富，编程、以及产品设计的见解。他从一个技术人员和创业者的视角去审视一个行业的发展，他的那些预测未来的观点再后来被一一的证实，并且给人类带来很多的好处。

作者从财富分配这个事情上面出发，讨论了我们应该怎么样的获得更多的财富，作者直接了当的说：就是通过创业来获得财富。作者认为财富是可以通过实践被创造出来，财富并不是一个零和的游戏，并不是我增加了你就必须减少。

在作者看来，一个人的能力和贡献在大公司里面很容易就会被平均化，要想让自己的能力显得突出，最好就是选择一个 10 人以内的小团队，这样一个人的能力和贡献才不会被平均化。

在这本书中，还有很多值得我们去认真品味的观点，虽然这本书早上 2004 年出版英文版，但是现在看来书上面的观点并不会随着时间的推移而不合时宜。

突破壁垒

发表于 2016-02-18 | 分类于 学习札记 |

如果不能控制这种逃避的倾向，再多、再好的任务管理技巧都是无效的，因为任务中相当重要的一部分（通常因为重要而困难，也因为困难而重要）永远无法完成。所谓效率，是在任务完成之后才能够衡量的。这样看来，对逃避困难>的人来讲，因为没有完整地完成任务，所以无论他们花了多长时间，也都没有效率（相当于分子为零）。

这是李笑来在他的《把时间当朋友》这本书上面第三章中提到的观点，讲的就是：当我们做一件事情的时候，遇到困难的时候，如果直面困难，我们的效率才能说是高的，否则就是一个效率底下的人。其中的观点在书中已经讲的非常明白了。看了他的观点，让我想起来前段时间我在学习 Python 的时候发生的一些事情。

我是在去年的时候开始学习 Python ，一直没有买书本，但是看了《简明 Python 教程》和《笨方法学习 Python 》，还有廖雪峰的《Python2.7 教程》我也看过了一点。看玩这些书本的时候，我对于 Python 的基本语法已经是比较了解了，可以写一些简单的爬虫。但是每当我要写一个比较大的东西，比如 Blog 网站的话，我就会在一些地方被难住了，有的困难经常就是困扰我好几天。面对这些情况，我选择重新学习那些基本的语法，再回头看语法的时候，其实我都能理解，但是再去写的时候就会不知道怎么实现。如此的恶性循环，我甚至觉得我是学不会编程的。

直到现在我看了李笑来的这本书才发现，可能原因并不是我真的不会编程，而是我在这样的一个困难面前没有跨过去，也就是常说的「壁垒」。或者当时的我跨过了这个壁垒，解决了这个困难，那我学习编程的道路就会明朗很多，不会像当初那么糟糕。

有些东西会让我们得心应手，这些不重要，重要的是那些阻碍我们前进的壁垒，只要突破了这些壁垒，我们就能看到新的道路。

这鸡汤，干了，共勉！

正确的转发朋友圈

发表于 2016-02-18 | 分类于 生活随笔 |

我经常会在朋友圈上面转发一些自己觉得有价值的东西，并且可以得避免自己转发一下谣言文章，并且在自己的朋友圈看到谣言性的文章都会提醒好友。

其实要想在朋友圈更加的有价值并不难，首先是从从自己做起，下面是我的几点建议，仅供参考：

文章来源信息

看到一篇文章，我们应该第一时间看一下这篇文章是怎么来的，包括看公众号信息、作者信息；是转载的还是原创？转载的话有没有备注出处？如果这些信息都都是非常模糊不清的话，那么这些文章就不应该转发出去，哪怕这是真实的情况，也会侵犯作者的版权问题。

文章内容

现在谣言穿的最多的就是关于小孩拐卖、养生、支付宝、微信账户信息的，每当看到这些信息的时候就应该提高自己的警惕，看一下这些做大的事情有没有被大型媒体或者身边的亲身经历人经历过，如果没有，那最好也是不要随意转发。

转发信任度较高的信息源

对于那些大型媒体或者一些比较有影响力的大 V 的文章，一般来说都是比较真实，不会放下太大的错误，但是错误还是会有的。虽然一些大型媒体来说也会发一些公关文，但是总比传播谣言要好。

之前我在界面的公众号看过了一篇名为「炸裂！《夏洛特烦恼》居然全片抄袭了《教父》导演的旧作！」的文章，觉得这篇文章还是挺有意思的，所以就转发了这篇文章。没过几天，在别的地方看到说到这篇文章是谣言来的，后来我查了一下，其实不能算是谣言，但是又不能说不是谣言，抄袭的地方还是会有的，只是不能判为全抄袭。

在上面的这个事例中我看到了其实谣言离我们还是挺近的，所以我们更加需要停下来认真的观察清楚这些文章的真实情况。但是我还是会遵守我的那个转发规则：公众号主体明确；标明作者；最重要的还是要符合科学常识。

药品招标如何影响药品的价格

发表于 2016-02-17 | 分类于 移动医疗 |

最近在关注一些关于药品招标和药品价格的问题，其实药品招标只是药品流通的一种形式而已，最初设定这个药品招标政策的目的就是为了患者可以在医院里面买到廉价的药物，但是经过这么多的药品招标，药价并没有实实在在的降下来，那么原因究竟是为什么呢？

首先来了解一下「药品招标」究竟是什么回事。简单来说「药品招标」就是需要采购药品的机构（医院）向符合条件投标人（卖药的，也就是药品代理公司，俗称药贩子）发布当年或者当季要采购的药品名称、数量等信息。投标人就会设定价格和优惠情况给采购机构，采购机构觉得符合自己意愿的采用和该投标人进货。这个过程中投标人可以使用多种方式让采购机构在他那里进货，其中的手段就不一一说明白了。

从中可以简单的看明白了，一个药品要想流通到患者的手里，那么必须要就过三个流程，第一个就是厂家到代理商；第二个就是代理商到医院；第三个就是医院在把药买个患者。先从第一层开始说起，厂家给到代理商的药品加成我们简单的称之为「出厂价」，这个价格基本上就是厂家核算过自己研发、生产、储存、物流等等费用之后得出来的价格。代理商给医院的价格我们称之为「招标价」，「招标价」和「出厂价」肯定是存在差价的，不然代理商吃西北方，至于这个差价又是多少呢，待会再来讨论。最后医院给到患者的药品价格有可能是比「招标价」要高，为什么是有可能呢？因为在过去，医院里面基本都是依靠「药品加成」来给医院创收的，但是最近两三年，有一部分医院已经取消了药品加成。所以那些取消药品加成的医院的药价基本就是医院进货时候的「招标价」了。

其实药厂和最后医院的「药品加成」都不是我们批评药品招标的点，我们要批评的靶子就是「招标价」于「出厂价」之间的差价。曾经多次被媒体报道出来「招标价」比「出厂价」平均高出 5 倍，有的甚至是高出 30-40 倍，这里面的利润都被医药代理公司瓜分已经在招标过程中进行暗箱操作。

这让我想起了之前取消药品限价之后药品升价的新闻，很多普通人在这个时候会把矛头指向药厂，认为这是药厂生产的药品，其中里面的利润肯定是药厂吃了。其实原因并不是这样子的，在调查发现，其中原因有很多，一个是因为原料药涨价，还有就是在过去限价的时候，药厂基本都是硬着头皮去生产这些药品，现在取消限价，涨价也是无可厚非的。其实还有一个原因就是代理公司有囤货现象，但是很多人都看不多这一面，只是知道药厂一定就是奸商。就好比春运火车票被黄牛党囤起来了，就把矛头指向中国铁路局。

关于国内药品流通的问题，还是有很多东西值得我们去认真思考的，或者我的这些思考都是错的。
