获取医药市场数据的途径（带链接）

发表于 2014-08-16 | 分类于 移动医疗 |

本文大部分信息来源新浪博客，具体出处不详，因看到没有添加链接，故添加上去。

1. 产品信息
产品批文国内产品数据来源：药监局网址-数据查询-公众查询/专业查询；中国药品注册数据库。国外产品数据来源：美国FDA；欧盟EMA；公司官网（导航网站）；

评点：以前药监局数据下载较难，于是偏好用中国药品注册数据库下载和查询。现在药监局开放公众查询，独家剂型和规格查看相对一目了然。不足之处是数据联动更新较慢，部分数据在专业查询中能体现，但公众查询中不能体现。

2. 产品在研情况
国内在研数据来源：丁香园；CDE；MIMS；药智数据；米内网 ；医药地理；公司年报；证券公司行研。

国外在研数据来源：美国FDA；欧盟EMA；公司年报/官网；证券公司行研；科研杂志。

评点：CDE越来越难破了，还要U-key，然后丁香园也不免费了，其他的都会按会员等级收费的。

3. 产品说明书
国内数据来源：药监局网址-数据查询-OTC说明书；药典有部分相关内容。其实最好的还是直接用丁香园的用药助手，而且有手机端的app

国外数据来源：MIMS、Drugs.com、美国FDA；、欧盟EMA。

评点：我国官方公布说明书的只有OTC药。平常查相关说明书要靠度娘本身嘈点就很多了，所以推荐使用丁香园的用药助手。

4. 医保信息
全国医保数据来源：人社。

各省增补医保数据来源：各省人社。

5. 基药目录信息
全国基药目录数据来源：卫计委，药监局-数据查询。

各省增补医保数据来源：各省卫计委。

评点：2012年版520国家版基药懂了吧，2013年省增补基药也懂了吧，那哪些省基药招标还要看2010年省增补知道吗？

6. 其它

  1. 是否OTC，OTC甲类/乙类？
    数据来源：药监局。

  2. 不良反应？
    数据来源：药监局通报；CNKI数据库

  3. 生产厂家更换？
    数据来源：国内外上市企业公告；米内网；IMS；微信微薄各种小道消息。

  4. 抗生素管理等级？
    数据来源：各省卫计委。

6. 价格

  1. 最高零售价
    数据来源：发改委；华招 ；中国医药网。
    评点：含单独定价，一般定价。主要医保产品。

  2. 地方物价
    数据来源：地方物价局；华招 ；中国医药网。
    评点：发改委通知再下达，地方增补医保产品。

  3. 中标价
    数据来源：各省采购平台；华招。

  4. 技术分公司规模
    数据来源：工信部。
    评点：目前分为基药和非基药招标，数据公开不公开看各省喜欢，数据难不难下载更是各省喜欢。此外，加入各省招标群也是获得数据的一种办法。

7. 市场规模
国内免费数据：IMS公开数据；米内网；公司年报；证券公司行研；雪球；辅料中国论坛。

国内收费数据：IMS，米内网。

国外免费数据：IMS公开数据；Drugs.com；公司年报；雪球；证券公司行研。

国外收费数据：IMS。

评点：国内的抽样数据，如果拿来考核，先弄清楚第三方布点哪家，然后下死命令开发，第三方数据辅助证明目前市场占有率真高。但是二级医院/县级医院没布点的真的不值得投资吗？

行研数据以前很喜欢慧博，现在慧博也收费了，就在慧博那里搜索，然后度娘下载。

8. 产品策略信息

  1. 产品ppt：豆丁；百度文库；微盘；爱问；公司网站等。诚意推荐冰点下载。

  2. 指南&解读：中华学会/医师协会各分会网站；丁香园；NCCN在中国；医脉通；CNKI；微薄微信博客等医师解读；。

  3. DA：参加各分会年会收集。代理产品可参加药交会收集。

  4. 流行病学：CNKI，卫计委年鉴。

  5. 患者流：第三方调研。

  6. 处方分析（收费）：IMS、医工院上海医院。

评点：个人觉得不是在专业网站上面的文档并不是非常可靠，一般不会是自家人上传的！

以上就是各种坊间正规途径的数据来源。至于数据如何解读，每个人的判断不一，偶觉得要多和别人分享、沟通，才能挖掘到数据下面的秘密。

第三方的数据仅仅是作为公司现状和外界均值比较的参考值。需知业界各著名第三方平均20%的行业增长率是拍脑袋出来，因为市场规模的放大倍数是“统计推算”出来的，不同推算方案结果不一。有时候，看到有些产品的增长率翻倍，大可以大胆预测，那一定是该产品原有市场基数很低。通常这样大胆的预测笔者应证十之八九是正确的。

对于自己公司的数据应如何分析，如何结合第三方数据分析，不在本文研究范围之内。如有兴趣，请留言探讨。此处对开篇问题作一些补充，话说数据分析是一件很时髦的工作方式。于是偶经常听到这样的故事：某童鞋拿到某第三方的数据，兴冲冲地在目标地图脑补这省份此类药市场潜力大哦，明年可以做目标市场哦。结果老板告之，此处市场准入规则不好玩公司要弃标。此案例说明：数据只能辅助决策，而不能完全取代决策。实际执行中，思考层面越多，数据越精准，判断可能越稳妥。

数据获取实质上没有啥秘笈，关键是要查得勤快且要善用Google和百度等搜索引擎。

对国内医药数据的期望？期望规格能统一格式，100mg 与 0.1g 可否统一。通用名统一格式。各省的医院有标准化的数据，级别能有数据可查。还有，统方合法化。
