正式使用 Hexo 搭建博客

发表于 2017-08-12


最近一两年，越来越多的人使用 Hexo 搭建博客，有其是有了 Next 这个主题之后，更是一发不可收拾。我用了 3 年的 WordPress 博客，也坚持下来写作，但是 WordPress 有几个缺点，比如系统庞大，一般的虚拟主机运行慢。找不到让我满意的主题，尝试过国外的付费主题，还是不太满意。官方的主题有很多问题，比 MarkDown 渲染效果很差。其实最主要还是自己不会开发出来更好的主题。

所以，Hexo 来了，今天折腾好几个小时，总算是完成，过程中有太多的坑，躺下去，一段让我想要放弃，最后还是重拾信心，算是完成吧。

至于 Hexo 这样的发布形式是不是符合写作习惯，有待考察。
