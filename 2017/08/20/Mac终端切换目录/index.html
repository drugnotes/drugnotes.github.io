Mac 终端切换目录

发表于 2017-08-20 | 分类于 学习札记 |

最近经常使用到 Mac 终端进行部署环境和安装包，需要给 Anaconda 里面安装jieba库的时候，发现 Anaconda 打开经常出错，所以只能使用终端进行安装。在终端里面需要打开硬盘下 anaconda/lib这个目录，然后执行 pip inatall jieba这个命令。打开终端，发现默认地址是我的 user 下的目录，使用ls进行查看的时候，发现并不是硬盘下面的目录。在网上找到一些关于 Mac 终端命令的文章，很多文章都没有提到怎么从user的跟目录切回到硬盘根目录。

找啊找，最后终于找到一篇文章提到这个切换的方式。

其实硬盘的根目录，就是user根目录的上一级目录，所以我们需要找到的是怎么切换回去当前目录的上一级目录，那篇博文中提到：

cd 切换工作目录
. 表示当前目录
.. 表示当前目录的上一级目录
/ 根目录/目录分隔符
./ 当前目录
../ 回到上一级目录
相对路径/绝对路径 凡是以/开始的路径，都是绝对路径
cd .. 进入上一级目录
cd 加波浪符号：进入当前登陆用户目录下

根据上面介绍，我们需要返回上一级目录，只需要输入cd ..即可。

虽然是一个很简单的问题，但是对于新手来说，这个问题就就如同前进道路的一座小山，只有翻过去才能继续前行。
