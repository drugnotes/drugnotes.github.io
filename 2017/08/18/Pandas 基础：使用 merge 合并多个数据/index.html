Pandas 基础：使用 merge 合并多个数据

发表于 2017-08-18 | 分类于 学习札记 |

Python 的 pandas 库，经常需要用到 merge 进行多个数据的合并，使用 merge 进行数据合并是横向的，pandas 还有一个函数是 cancat ，也是用于多个数据的合并，但是 concat 是纵向合并，merge 是横向合并。

merge 基础合并操作中，需要掌握的有下面几点：

  1. 依据一组 key 数据进行合并；
  2. 依据两组或者两组以上的 key 进行合并；
  3. 依据 index 进行合并；
  4. 合并的四种方式，分别是 inner、outer、left、right；
  5. 显示合并的方式，也就是 indicator 参数；
  6. 合并时，表头重名时的处理方法，也就是使用 suffixes 生成别称。

好吧，下面就跟着代码一起学习学习

依据一组 key 数据进行合并

首先，生成两个Dataframe，以一组key为合并依据，合并两列数据

依据两组或者两组以上的 key 进行合并

除了单个key进行合并之外，merge 还支持多组key进行合并，比如两组 key （key1和key2）为依据进行合并

依据 index 进行合并

除了支持一组和多组key进行合并外，还支持index为依据进行合并

合并的四种方式，分别是 inner、outer、left、right

How参数声明合并方式，4 中合并方式，分别是inner、outer、left、right。默认情况下，使用inner，只有 key 值相同的行才合并

使用outer，将每一行字段进行合并，没有的部分使用NaN填充

使用left，以左边的 DataFrame 条件（key)为准合并，左边没有的去掉,类似SQL语句中的左连接。

使用right，以右边的 DataFrame 条件（key)为准合并，右边没有的去掉，类似SQL语句中的右连接。

显示合并的方式，也就是 indicator 参数

merge 还提供indicator参数，用途就是在合并数据中生成新的一列，名字为_merge，用于声明每一行合并时的规则

当然，你还可以对新生成的_merge列进行新的命名

合并时，表头重名时的处理方法，也就是使用 suffixes 生成别称

在合并中，两个DataFrame中的表头信息相同时，需要使用到suffixes这个参数，比如两个DataFrame中两列都是a,b,就可以在生成新表的时候生成别名
